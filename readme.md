# GREEN-API Тестовое задание (DevOps Developer)

Интерфейс для взаимодействия с методами GREEN-API, реализованный в рамках тестового задания.

## Особенности реализации:
* **Zero-dependency**: Только чистый HTML/JS без тяжелых фреймворков.
* **Observability**: Добавлен замер Latency для каждого запроса к API.
* **CI/CD**: Настроен автоматический деплой через GitHub Actions на GitHub Pages.
* **Security**: Параметры инстанса не хардкодятся в коде, а вводятся пользователем в рантайме.

## Как запустить:
1. Проект опубликован по ссылке: [ТВОЯ_ССЫЛКА_ПОЯВИТСЯ_ЗДЕСЬ]
2. Введите свои `idInstance` и `apiTokenInstance` из личного кабинета GREEN-API.
3. Протестируйте методы отправки сообщений и получения настроек.
